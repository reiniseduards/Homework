# Homework

How to use:
* Open Homework\bin\Debug\net6.0-windows\WpfApp1 - Solution was placed on "C:\" while developing, but should work from anywhere.
* Load the given CSV file
* Load the given XML file - file was saved localy as .xml)
* Press "Export"
The solution then creates a dated "output.csv" file located in the folder "output" and a dated "log.txt" file located in "output\logs"
